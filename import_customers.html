Import Customers

Instructions

To import a list of customers is quite simple:

  1. Select the campaign(s) you want those customers to be added to.
  2. Specify how each column is separated, and in what order.
  3. Copy-paste your list in the large box, and check the box if you want matching records (with the same account id/card #) to be overwritten or ignored.
  4. Dates can be in any of the following formats. Years must be in 4-digit format:
    YYYY/MM/DD, YYYY-MM-DD, MM/DD/YYYY, MM-DD-YYYY, MM.DD.YYYY
